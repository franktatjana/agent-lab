# Networking Agent

Helps build, maintain, and leverage professional relationships strategically, turning networking from an awkward obligation into an intentional practice.

## What it does

The Networking Agent guides you through assessing your current network, reaching the right people, maintaining relationships over time, and asking for help when you need it. It treats networking as a learnable craft that works for any personality type.

## When to use it

- Starting in a new role or industry and need to build connections
- Have a strong network but neglect it and relationships go cold
- Need to reach a specific person but don't know how to approach them
- Want to ask for introductions but feel awkward about it
- Looking for a systematic approach to relationship management

## Skills

| Skill | What it does |
|-------|-------------|
| **Network Audit** | Map your network, find gaps, identify who to connect with |
| **Outreach Campaign** | Research, craft messages, and prepare for conversations |
| **Relationship Maintenance** | Follow-up cadence, touchpoints, and visibility building |
| **Introduction Strategy** | How to ask for and give warm introductions |

## Personalities

| Personality | Best for |
|-------------|----------|
| **Connector** | Warm, relationship-first approach. Genuine connections over tactics. |
| **Strategist** | Analytical, efficient. Career-focused networking with clear ROI. |
| **Coach** | Patient, encouraging. Building confidence and long-term habits. |

## Frameworks

- Weak Ties Theory (Granovetter)
- Give First Approach (Adam Grant)
- Dunbar's Numbers (5/15/50/150)
- Network Mapping
- Personal CRM
