# Find Gaps

Identify the most significant cultural gaps between two or more cultures.

**Input:** {cultures}, {context}

**Output:** Ranked list of gaps with practical implications

---

Compare {cultures} and identify the most significant gaps for {context}:

For each major gap:

1. **Dimension**: Which cultural dimension?
2. **Positions**: Where does each culture fall?
3. **Gap size**: How far apart? (Large/Medium/Small)
4. **Practical impact**: How will this affect interactions?
5. **Common misread**: How might each side misinterpret the other?

**Priority gaps** (rank by impact on {context}):

1. [Most impactful gap]
2. [Second most impactful]
3. [Third most impactful]

**Surprising similarities:**

Also note any dimensions where these cultures are unexpectedly aligned, this is useful for building common ground.

Be specific to these cultures. Avoid generic cultural advice.
