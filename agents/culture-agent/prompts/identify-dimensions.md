# Identify Dimensions

Map a culture to Hofstede and Meyer framework dimensions.

**Input:** {culture}, {context}

**Output:** Dimension scores with explanations

---

Analyze {culture} across cultural dimensions for {context} context:

**Hofstede Dimensions:**

1. Power Distance (0-100): How is hierarchy perceived?
2. Individualism (0-100): Individual vs. group orientation?
3. Masculinity (0-100): Competition vs. cooperation?
4. Uncertainty Avoidance (0-100): Comfort with ambiguity?
5. Long-Term Orientation (0-100): Future vs. present focus?
6. Indulgence (0-100): Gratification vs. restraint?

**Meyer Culture Map:**

1. Communicating: Low-context ←→ High-context
2. Evaluating: Direct negative feedback ←→ Indirect
3. Persuading: Principles-first ←→ Applications-first
4. Leading: Egalitarian ←→ Hierarchical
5. Deciding: Consensual ←→ Top-down
6. Trusting: Task-based ←→ Relationship-based
7. Disagreeing: Confrontational ←→ Avoids confrontation
8. Scheduling: Linear time ←→ Flexible time

For each dimension:
- Provide score or position
- Explain what this means in practice for {context}
- Note regional/generational variations if significant

Output only the cultural profile, no additional commentary.
