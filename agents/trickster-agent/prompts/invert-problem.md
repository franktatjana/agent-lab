# Invert Problem

Flip the problem upside down, inside out, and backwards. What if the opposite were true? What if the "problem" were the solution? What if the direction everyone is pushing is exactly wrong? Systematic inversion reveals the hidden structure of how the user has framed their situation.

**Input:** {stuck_situation}, {invisible_assumptions}
**Output:** Array of inversions, each with original_frame, inverted_frame, absurd_solution, real_insight_hidden_inside

---

Take the stuck situation and the invisible assumptions that have been surfaced. For each assumption and for the problem itself, perform a deliberate inversion.

Inversion techniques:

1. **Opposite day**: State the exact opposite of the assumption. "We need to slow down to improve quality" becomes "We need to speed up to improve quality." Does the opposite contain any truth?
2. **Problem-as-solution**: What if the thing they are trying to eliminate is actually the thing they need more of? "We have too many meetings" becomes "What if the problem is we don't have the right meetings?"
3. **Reverse causality**: Flip the cause and effect. "Technical debt slows us down" becomes "Slowness creates technical debt." Which direction is actually causal?
4. **Swap the subject**: What if the wrong entity is doing the work? "The team needs to adapt to the process" becomes "The process needs to adapt to the team."
5. **Negate the goal**: What if achieving the stated goal would actually make things worse? "We need to grow revenue" becomes "What if growing revenue right now would destroy us?"

For each inversion:
- State the original frame clearly (how the user currently sees it)
- State the inverted frame (the flip)
- Propose one deliberately absurd solution that follows from the inverted frame
- Extract the real insight hiding inside the absurdity

The absurd solution is not the answer. It is a magnifying glass that makes the real insight visible. Name the real insight explicitly.

Do NOT:
- Flip words without flipping meaning (mechanical inversion is not useful)
- Propose absurdity for shock value without a real insight underneath
- Invert things that are genuinely fixed constraints (physics, law, etc.) without flagging you are doing so
- Present inversions as truth (they are lenses, not conclusions)
- Generate more than 3 inversions (depth over breadth)
