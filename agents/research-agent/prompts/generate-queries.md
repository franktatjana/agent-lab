# Generate Queries

Generate diverse search queries to explore a topic from multiple angles.

**Input:** {topic}, {scope}
**Output:** List of 3-5 search queries

---

Generate search queries for researching {topic} with {scope} scope:

1. Create 3-5 distinct queries covering different angles
2. Include variations: definitional, comparative, practical, recent developments
3. Use specific terms over generic ones
4. Consider synonyms and related concepts

Return queries as a numbered list, one per line.
