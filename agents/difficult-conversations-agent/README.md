# Difficult Conversations Agent

Prepares for and navigates challenging interpersonal conversations. Separates the layers that make conversations difficult, then addresses each one with the right framework.

## When to Use

- Delivering bad news (layoffs, missed promotions, project cancellations)
- Addressing performance issues before they become crises
- Resolving conflict between colleagues or teams
- Raising sensitive topics with a manager or stakeholder
- Repairing a relationship after a conversation went wrong

## Skills

| Skill | What it does |
|-------|-------------|
| Conversation Preparation | Maps the situation, chooses a framework, and crafts an opening statement |
| In-Conversation Toolkit | Pre-built responses for likely reactions plus self-regulation techniques |
| Relationship Repair | Post-conversation follow-up plan to maintain the relationship |

## Personalities

| Personality | Best for |
|-------------|----------|
| Empathetic | Sensitive situations, fragile trust, high emotional stakes |
| Direct | Time-sensitive feedback, when clarity is kindness |
| Mediator | Conflict resolution, neither party clearly right |

## Frameworks

- Difficult Conversations Model (Stone, Patton, Heen)
- Nonviolent Communication (Rosenberg)
- Crucial Conversations / STATE method (Patterson et al.)
- DEAR MAN (DBT)
